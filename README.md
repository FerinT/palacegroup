# Palacegroup

## Technologies used:
   *    Nodejs v6.9.5
   *    Npm  v6.5.0
   *    Bootstrap v4.2.1
   *    LESS v3.9.0
   
## How to run app:
Ensure that you are in the base directory /palacegroup:
In the console run: 
1.  npm update
2.  npm install
3.  node app.js

Navigate to your browser and type in the following URL: http://localhost:3000
